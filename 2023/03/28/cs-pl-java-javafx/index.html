Application

Application 是抽象类，需要重写抽象方法 start，静态函数 lanuch 依次调用 init， start，stop。

  • 添加一个按钮，链接到网站
  • 按钮打开新的场景

Stage

函数：

  • setTitle()
  • getIcons().add(new Image("image/image.jpg"))：设置图标
  • setResizable()：窗口大小是否可变
  • initStyle(StageStyle)：设置窗口样式（四种）
  • initOwner：设置父窗口
  • initModality()：设置模态，默认其它窗口可用
    • APPLICATION_MODAL：除当前窗口外其它窗口不可用
    • WINDOW_MODAL：只有父窗口不可用，打开的其它窗口可用

事件：退出程序

Scene

setScene：切换场景

setCursor(new ImageCursor())：设置鼠标光标图片

Node

Node 是抽象类，不能被实例化，所有可见的控件继承 Node

通用属性：

  • setLayoutX，setLayoutY：设置坐标，默认为 000 （用 AnchorPane 布局）
  • setTranslateX，setTranslateX：横竖平移
  • setPrefWidth，setPrefHeight：设置边框的长、高
  • setAlignment(Pos.CENTER)：文字居中
  • SetStyle：设置样式：边框、背景等（CSS 语法）
    • -fx-background-color
    • -fx-border-color
    • -fx-border-width （宽度，传入像素）
  • setRotate：旋转
  • SetOpacity：000 到 111 之间的值，设置透明度
  • SetVisible：标签是否可见
  • blendMode：假设有红蓝两个框，重叠部分颜色色叠加

Circle

圆相关的属性设置

  • setFill：填充颜色
  • setStroke: 设置边框
    • setStrokeWidth：粗细

Canvas 画布

getGraphicsContext2D 相关函数：

  • setLineWidth
  • setStroke：边框颜色
  • strokeRect：矩形 （Oval：圆形）
  • setFill + fillRec：填充颜色
