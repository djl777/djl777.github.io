质数

常见结论和性质

  1. $\pi(n)$ 表示小于等于 $n$ 的质数个数，有

  1. 第 $n$ 个质数是 $nlogn$ 级别 $p_n = O(nlogn)$

  2. 调和级数相关

    • $\sum_{i = 1} ^ n \frac{1}{i} = O(logn)$
    • $\sum_{1 \le p \le n} \frac{1}{p} = O(loglogn)$

 

整除 公约数 公倍数

常见性质

$gcd(a,b)$ 记为 $(a, b)$，最小公倍数简记为 $[a,b]$

整除

  1. $a|c,b|c,(a,b) = 1 \Rightarrow ab|c$

    感性理解：$a$，$b$ 没有公共因子

  2. $a|bc,(a,b) = 1 \Rightarrow a|c$

  3. $p|ab \Rightarrow p|a$ 或 $p|b$

公约数 公倍数

$a = p_1^{a_1}p_2^{a_2}…p_k^{a_k}$（$a_i$ 可能为 $0$），$b = p_1^{b_1}p_2^{b_2}…p_k^{b_k}$

$(a, b) = \Pi P_i ^ {min(a_i, b_i)}$，$[a, b] = \Pi P_i ^ {max(a_i, b_i)}$，$(a,b)[a,b] = ab$

 

欧几里得算法

$(a,b) = (a - b, b) = (a \mod b, b)$

  • 时间复杂度：$a \mod b$ 至少除以 $2$（分类讨论），复杂度为 $O(log \min(a, b))$
  • 一般的，对于多个数 $a_1, a_2…a_n$ 做欧几里得的时间复杂度是 $O(n + log \max{a_i})$

  • 另一种算法

    $a$，$b$ 都是奇数，$(a,b) = (a - b, b)$

    $a$ 是偶数，$b$ 是奇数，$(a,b) = (a / 2, b)$

    $a$，$b$ 都是偶数，$(a,b) = (a/2, b/2)$

    适用条件：

    • 高精度，取余操作比较慢
    • int128，long long 会快一点
