技巧

向上取整 $\lceil \frac{a}{b} \rceil$ （两种均可）

python 是标准的向下取整，而 c++ / go / java 等是向 $0$，因此当 $a = 0$ 时 $(1)$ 式会有问题，为了兼容统一用 $(2)$ 式

证明：

  • $a = kb$，显然
  • $a = kb + r$，$1 \le r \le b - 1$，因为 $a + b - 1 = kb + r + b - 1$， $b \le r + b - 1 \le 2b - 1$，右等于左等于 $k + 1$
