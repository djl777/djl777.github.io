向上取整 ⌈ab⌉\lceil \frac{a}{b} \rceil⌈ba​⌉ （两种均可）

⌊a−1b⌋+1⌊a+b−1b⌋\begin{align} \lfloor \frac{a - 1}{b} \rfloor + 1 \tag{1} \\ \lfloor \frac{a + b - 1}{b} \rfloor \tag{2} \end{align} ⌊ba−1​⌋+1⌊ba+b−1​⌋​(1)(2)​

python 是标准的向下取整，而 c++ / go / java 等是向 000，因此当 a=0a = 0a=0 时 (1)(1)(1) 式会有问题，为了兼容统一用 (2)(2)(2) 式

证明：

  • a=kba = kba=kb，显然
  • a=kb+ra = kb + ra=kb+r，1≤r≤b−11 \le r \le b - 11≤r≤b−1，因为 a+b−1=kb+r+b−1a + b - 1 = kb + r + b - 1a+b−1=kb+r+b−1， b≤r+b−1≤2b−1b \le r + b - 1 \le 2b - 1b≤r+b−1≤2b−1，右等于左等于 k+1k + 1k+1
