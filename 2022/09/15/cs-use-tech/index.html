软件

下载：

  • github 上，点进去 release 可以下载一些手机软件

 

coq

  1. 下载 coq
  2. vscode 上安装 vscoq
  3. 按照插件提升添加路径
  4. 具体技巧和快捷键，ctrl + shift + p，输入 coq 可查看

 

MSYS2，Termux

  1. 官网下载 MSYS2
  2. 手机下载 Termux （github 上）
  3. MSYS2 使用 pacman 系列命令
  4. Termux 使用 apt 系列命令
