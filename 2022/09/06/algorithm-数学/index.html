E - Add and Mex

给定一个数组，每次使整个区间分别加上对应下标的值，求数组的 mex 函数（数组里不存在的自然数的最小值）

  • 性质：mex 函数的最小值是 $0$，最大值是 $n$，一个数加上某一个值如果小于 $0$ 或者大于 $n$，那么它对答案一定不会做出贡献
  • 第一项最多贡献 n，第二项n/2，…. 第n项1，调和级数nlogn
  • 只需要加入 0 - n-1 以内的数

4622. 整数拆分)

数论 哥德巴赫猜想

规定 $f(x)(x≥2)$ 表示整数 x 除本身外的最大因数。将 x 进行拆分，求最小 $f(x)$ 和

  • 任一大于2的偶数都可写成两个素数之和
  • 任何一个大于7的奇数都能被表示成三个奇质数的和
  • 一个奇数若分解成两个，则必含2

4623. 买糖果)

数论

给一个总钱数 T，若干个糖果店，顺时针每次买一个，如果可以买则买，否则去下一个，求购买的糖果数

  • 看似是 $o(n^2)$ ，实际上，设每一圈购买的总钱数为 sum ，则有 $T \% sum < T/2$ （分 sum 与 T/2 的关系讨论即可），时间复杂度为 $o(nlogn)$

365. 水壶问题 （经典面试题）

数论

  • 两个杯子不可能同时为空，或同时不满
  • 把两个杯子看成一个整体，它与外界的交换只有4种情况：+a，-a，+b，-b
    • 不存在一个杯子在半满的情况下与外界交换（性质1易证）
  • 问题转化为 $ax + by = c$，其中 x, y 为整数，由裴蜀定理，有 $gcd(a, b) | c$
  • 构造：如果系数存在，且满足 $0 < c \le a + b$（$c = 0$ 直接返回），则必然有一个系数大于0，不妨设 $x > 0$，若 $y > 0$，此时必是 $c = a + b$；$y \le 0$，不断地加满 a，然后不断地往外倒 b

 

754. 到达终点数字

从 $0$ 开始，每次可选择向左或者向右移动 $i$ 步，$i$ 从 $1$ 开始逐渐递增，问移动到 $target$ 的最小步数

 

对于 $target < 0$ 的情况，只需要将 $-target$ 的步骤取反，因此只考虑 $target > 0$

给出算法：找到最小正整数 $n$，满足 $s = \Sigma_{i = 1}^ni \ge target$ 且 $s$ 的奇偶性与 $target$ 相同

  1. 证明满足上述两个条件时，$target$ 能被合法得到

$s = target$ 时，显然成立

$s > target$ 时，首先证明：

引理：设 $s = \Sigma_{i = 1}^ni$，$[1,s]$ 内的每个数都可以由若干个不同的 $[1,n]$ 的数组成

证明：数学归纳，$n = 1,2$ 时显然，设 $n \le k$ 时结论成立，可以用 $[s-k,s]$ 内的数字加上 $k + 1$，得到 $[s + 1, s + k + 1]$，从而 $n = k + 1$ 成立

因此，我们可以从中挑出一些步数变为向左走（即将正号变为负号，不妨设为 $a_1,a_2…a_k$），需要满足 $s - 2(a_1 + a_2 + … a_k) = target$，即 $a_1 + a_2 + … a_k = \frac{s - target}{2}$

由于 $1 \le \frac{s - target}{2} \le s$，由引理可知，这样的 $a_i$ 组合一定存在

  1. 最优性

显然需要满足 $s’ = \Sigma_{i = 1}^{n’}i \ge target$，否则无法到达，我们找到第一个满足 $s’ \ge target$ 的 $n’$

  • $s’ = target$ ，$n’$ 显然是最小步数（情况一）

  • $s’ > target$

    • $s’ - target$ 为偶数，$n’$ 显然是最小步数（情况二）
    • $s’ - target$ 为奇数，由 $1$ 的证明，此时一定不能通过反向变为 $target$，因此需要添加步数
      • 添加一步后（+奇）相距变为偶数（情况三）
      • 添加一步后相距仍是奇数（+偶），两步后相距变为偶数（+奇)（情况四）

补充证明：会存在上述情况且仅会出现上述情况

$s = \frac{k(k+1)}{2}$

$k = 4X$，$s = 2X(4X + 1)$，为偶

$k = 4X + 1$，$s = (2X+1)(4X + 1)$，为奇

$k = 4X + 2$，$s = (2X+1)(4X + 3)$，为奇

$k = 4X + 3$，$s = (2X + 2)(4X + 3)$，为偶

再根据 $target$ 的奇偶性不难得到上述情况
