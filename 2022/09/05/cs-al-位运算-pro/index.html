2397. 被列覆盖的最多行数

给一个二进制矩阵 mat 和一个整数 cols ，表示你需要选出的列数。

如果一行中，所有的 1 都被选中的列所覆盖，称这一行 被覆盖 了。

返回在选择 cols 列的情况下，被覆盖 的 最大 行数。

  • 1≤mat.length,mat[i].length≤121 \le mat.length,mat[i].length \le 121≤mat.length,mat[i].length≤12
  • 1≤cols≤n1 \le cols \le n1≤cols≤n

 

二进制枚举

 

Gosper’s Hack

二进制枚举 n 列中 col 列为 1 的所有数，以 col = 4, n = 7 为例，最小为 0001111，最大为 1111000，该算法能在 o(1)o(1)o(1) 的时间求出符合要求的下一个数：

  1. 对于其中任意一个数，它的下一个数为找到第一个出现01的地方，将其换成10，然后将右边的所有1移到末尾(0101110 --> 0110011)
  2. x+lowbit(x)x + lowbit(x)x+lowbit(x) 为数的左半部分(0101110 + 10 = 0110000)
  3. x⊕(x+lb)/lb>>2x \oplus (x + lb) / lb >> 2x⊕(x+lb)/lb>>2 为数的右半部分(0011110 --> 001111 --> 0011)
