java 基本语法

发表于2022-08-24|更新于2023-04-01|计算机编程语言java
|字数总计:1.1k|阅读时长:4分钟|阅读量:

基础

输入输出

效率较高，输入规模较大时使用。注意需要抛异常。

 

数组

数组不可变长

  • a.hashcode() ：返回对象物理地址的哈希值

  • a.fill(int[] a, int val)：填充数组

  • a.toString()：将数组转化为字符串

  • a.deepToString()：将多维数组转化为字符串

  • 自定义排序

 

String

只读变量，不可变

  • a.hashcode() ：重载，返回字符串的哈希值
  • Integer.parseInt(a)：字符串转整数（其它类似）
  • charAt(i)：访问字符
  • indexOf(char c)、indexOf(String str)：查找，找不到返回 −1-1−1
  • compareTo()：判断两个字符串的字典序大小，负数表示小于，000 表示相等，正数表示大于
  • startsWith()、endsWith()：判断是否以某个前缀开头、某个后缀结尾
  • trim()：去掉首尾的空白字符
  • toLowerCase()、toUpperCase()：全部用小写、大写
  • replace(char oldChar, char newChar)：替换字符
  • replace(String oldRegex, String newRegex)：替换字符串
  • replaceAll()
  • substring(int beginIndex, int endIndex)：返回 [beginIndex, endIndex) 中的子串

StringBuilder、StringBuffer

可修改，StringBuffer 线程安全，速度较慢；StringBuilder 线程不安全，速度较快。

  • reverse()：翻转字符串

 

函数

Java 的所有变量和函数都要定义在类中。

函数或变量前加 static 表示静态对象，类似于全局变量。静态对象属于 class，而不属于 class 的具体实例。访问时用类名。

静态函数中只能调用静态函数和静态变量。

 

类与接口

  • 一个源文件中只能有一个 public 类，可以有多个非 public 类。
  • 源文件的名称和 public 类的类名保持一致。
  • 同一个目录下的类不需要 import

类的多态

 

接口

  • 实现多人协作，定义的函数需要实现

  • 每个接口可以继承多个接口

  • 每个类可以实现多个接口

  • 接口多态见示例

 

泛型

泛型只能使用对象类型，不能使用基本变量类型。

 

常用容器

List

接口：java.util.List<>

实现：

  • java.util.ArrayList<>：变长数组
  • java.util.LinkedList<>：双链表

函数：

  • add()：在末尾添加一个元素
  • get(i)：获取第 i 个元素
  • set(i, val)：将第 i 个元素设置为 val

stack

队列

接口：java.util.Queue<>

实现：

  • java.util.LinkedList<>：双链表
  • java.util.PriorityQueue<>：优先队列
    • 默认是小根堆，大根堆写法：new PriorityQueue<>(Collections.reverseOrder())

函数：

  • remove()：删除并返回队头

Set

接口：java.util.Set<K>

实现：

  • java.util.HashSet<K>：哈希表
  • java.util.TreeSet<K>：平衡树

函数：

  • contains()：是否包含某个元素
  • remove()：删除元素

Map（类似Set）

  • Map.Entry<K, V>：Map 中的对象类型

    • getKey()：获取关键字
    • getValue()：获取值

  • getOrDefault(key, value)：默认值

  • merge(key, value, Integer::sum)：合并操作
