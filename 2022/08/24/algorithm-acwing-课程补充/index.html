多路归并

146. 序列

  • 合并 m 个序列不好想，考虑每次合并两个，保留 n 个元素，一共合并 m 次

  • 对于两个序列：

    a1 a2 a3 … an （排序）

    b1 b2 b3 … bn

    分组：

    b1 + a1, b1 + a2 … b1 + an

    b2 + a1, b2 + a2 …

    …

    每组内部是有序的，每次挑选一个数，将组内下一个数加入

  • 时间复杂度 o(mnlogn)o(mnlogn)o(mnlogn)
