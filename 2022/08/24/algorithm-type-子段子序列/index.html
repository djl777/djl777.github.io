删除操作

2382. 删除操作后的最大子段和

按照题目给定的顺序从数组（元素均为正整数）删除元素，求此时还剩下元素的最大子段和

  • $1 \le n \le 10^5$
  • $1 \le nums[i] \le 10^9$

 

并查集（逆向）

  1. 考虑逆过程，即把子段不断合并

  2. 新技巧：将 x 合并到 x + 1 ，把链看成是一列火车，这列火车有一节「幽灵火车头」，不算在实际的子段中。如果要合并两条链，就需要把左边这节幽灵火车头作为一节实际的车厢加到右边这列火车中。

  3. 对于 ans[i]，要么取上一个 ans[i+1] 的最大子段和，要么取合并后的子段和，这两者取最大值。

set 模拟（正向）

 

第k大/小和

2386. 找出数组的第 K 大和 (子序列)

给你一个整数数组 nums 和一个 正 整数 k 。你可以选择数组的任一 子序列 并且对其全部元素求和。

数组的 第 k 大和 定义为：可以获得的第 k 个 最大 子序列和（子序列和允许出现重复），求 第 k 大和 。

注意：空子序列的和视作 0 。

  • $n = nums.length$

  • $1 \le n \le 10^5$

  • $-10^9 \le nums[i] \le 10^9$

  • $1 \le k \le min(2000, 2^n)$

     

堆

  1. 考虑较简单的第 k 小子序列和问题：

    • 排序，建堆

    • 设一个二元组 $(sum,i)$，表示以第 i 位为结尾,，和为 sum 的子序列，每次取出堆顶，然后把 $(sum+a_i+1,i+1)$ 和 $(sum+ai+1−ai,i+1)$ 加入堆中, 即把第 i+1个数加入到序列中, 或把序列中最大的数替换成第 i+1个数，显然这样可以不重不漏地按顺序遍历完所有情况。（可以用数学归纳法证明，这样做可以构造出所有的子序列）

    • 第 k 个被取出堆顶的数就是答案

  2. 对于第k大，只要从最大的和减去最小

  3. 正数负数的处理
  4. 复杂度 $o(nlogn + klogk)$

二路归并

  1. 正数负数的处理同上
  2. 对于某个数，选或者不选两种情况，从 i-1 到 i，只需要保留前 k 个数作为备选
  3. 只需要考虑最小的 k 个数，反证，如果选择了k+1以后的数，则可以将它替换成前 k 个数中的任意一个数，结果不会变差
  4. 复杂度 $o(nlogn + k^2)$

 

P2048 [NOI2010] 超级钢琴

题意：求长度为 L 到 R 的前 k 大子段和

堆 + RMQ + 前缀和

  1. 定义 $MAX(o, l, r) = max{sum(t) - sum(o - 1) \ | \ l\leq t \leq r }$ ，即以 $o$ 为左端点，右端点范围是$[l, r]$的最大子段，则只需要$sum(t)$最大。问题转换为求$sum(t)$在$[l, r]$的最大值，使用RMQ
  2. 对于$(o, l, r)$，设其最优解位置是$m$，将它从堆顶弹出时，最优解还可能存在于 $t$ 左右的两端区间中，为了避免重复且不丧失其他较优解，仍然要把 $(o, l, t - 1)$ , $(o, t + 1, r)$ 放回。显然，在放回去之前应保证区间存在，即 $l = t$或 $r = t$ 的情况特判。
  3. RMQ 数组记录的原本是最大值，这里需要的是下标，对相关函数略作调整
