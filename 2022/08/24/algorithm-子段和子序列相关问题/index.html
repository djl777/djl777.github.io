2382. 删除操作后的最大子段和

给你两个下标从 0 开始的整数数组 nums 和 removeQueries ，两者长度都为 n 。对于第 i 个查询，nums 中位于下标 removeQueries[i] 处的元素被删除，将 nums 分割成更小的子段。

一个 子段 是 nums 中连续 正 整数形成的序列。子段和 是子段中所有元素的和。

请你返回一个长度为 n 的整数数组 answer ，其中 answer[i]是第 i 次删除操作以后的 最大 子段和。

注意：一个下标至多只会被删除一次。

示例：

提示：

  • n == nums.length == removeQueries.length
  • 1 <= n <= 10^5
  • 1 <= nums[i] <= 10^9
  • 0 <= removeQueries[i] < n
  • removeQueries 中所有数字 互不相同

 

并查集

  1. 考虑逆过程，即把子段不断合并

  2. 新的技巧：将x合并到 x + 1 ，把链看成是一列火车，这列火车有一节「幽灵火车头」，不算在实际的子段中。如果要合并两条链，就需要把左边这节幽灵火车头作为一节实际的车厢加到右边这列火车中。

  3. 对于 ans[i]，要么取上一个 ans[i+1] 的最大子段和，要么取合并后的子段和，这两者取最大值。

第k大/小和

2386. 找出数组的第 K 大和 (子序列)

给你一个整数数组 nums 和一个 正 整数 k 。你可以选择数组的任一 子序列 并且对其全部元素求和。

数组的 第 k 大和 定义为：可以获得的第 k 个 最大 子序列和（子序列和允许出现重复）

返回数组的 第 k 大和 。

子序列是一个可以由其他数组删除某些或不删除元素排生而来的数组，且派生过程不改变剩余元素的顺序。

注意：空子序列的和视作 0 。

提示：

  • n == nums.length

  • 1 <= n <= 105

  • -109 <= nums[i] <= 109

  • 1 <= k <= min(2000, 2n)

     

堆

  1. 考虑较简单的第 k 小子序列和问题：

    • 排序，建堆

    • 设一个二元组 (sum,i) ，表示以第 i 位为结尾,，和为 sum 的子序列，每次取出堆顶，然后把 (sum+ai+1,i+1) 和 (sum+ai+1−ai,i+1) 加入堆中, 即把第 i+1个数加入到序列中, 或把序列中最大的数替换成第 i+1个数，显然这样可以不重不漏地按顺序遍历完所有情况。（可以用数学归纳法证明，这样做可以构造出所有的子序列）

    • 第 k 个被取出堆顶的数就是答案

  2. 对于第k大，只要从最大的和减去最小

  3. 正数负数的处理

  4. 复杂度 o(nlogn+klogk)o(nlogn + klogk)o(nlogn+klogk)

二路归并

  1. 正数负数的处理同上
  2. 对于某个数，选或者不选两种情况，从i-1到i，只需要保留前k个数作为备选
  3. 只需要考虑最小的k个数，反证，如果选择了k+1以后的数，则可以将它替换成前k个数中的任意一个数，结果不会变差
  4. 复杂度 o(nlogn+k2)o(nlogn + k^2)o(nlogn+k2)

 

P2048 [NOI2010] 超级钢琴

题意：求长度为L到R的前k大子段和

堆 + RMQ + 前缀和

  1. 定义 MAX(o,l,r)=max{sum(t)−sum(o−1) ∣ l≤t≤r}MAX(o, l, r) = max\{sum(t) - sum(o - 1) \ | \ l\leq t \leq r \}MAX(o,l,r)=max{sum(t)−sum(o−1) ∣ l≤t≤r} ，即以 ooo 为左端点，右端点范围是[l,r][l, r][l,r]的最大子段，则只需要sum(t)sum(t)sum(t)最大。问题转换为求sum(t)sum(t)sum(t)在[l,r][l, r][l,r]的最大值，使用RMQ
  2. 对于(o,l,r)(o, l, r)(o,l,r)，设其最优解位置是mmm，将它从堆顶弹出时，最优解还可能存在于 ttt 左右的两端区间中，为了避免重复且不丧失其他较优解，仍然要把 (o,l,t−1)(o, l, t - 1)(o,l,t−1) , (o,t+1,r)(o, t + 1, r)(o,t+1,r) 放回。显然，在放回去之前应保证区间存在，即 l=tl = tl=t或 r=tr = tr=t 的情况特判。
  3. RMQ 数组记录的原本是最大值，这里需要的是下标，对相关函数略作调整
