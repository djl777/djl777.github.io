多路归并 146. 序列

mmm 个序列，每个包含 nnn 个非负整数，从每个序列中选一个元素相加，求序列和最小的 nnn 个值

 

  • 合并 mmm 个序列不好想，考虑每次合并两个，保留 nnn 个元素，一共合并 mmm 次

  • 对于两个序列：

    a1a_1a1​ a2 a3 … an （排序）

    b1 b2 b3 … bn

    分组：

    b1 + a1, b1 + a2 … b1 + an

    b2 + a1, b2 + a2 …

    …

    每组内部是有序的，每次挑选一个数，将组内下一个数加入

  • 时间复杂度 o(mnlogn)o(mnlogn)o(mnlogn)
