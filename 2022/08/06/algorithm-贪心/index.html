LeetCode

2375. 根据模式串构造最小数字

给你下标从 0 开始、长度为 n 的字符串 pattern ，包含两种字符，'I' 表示 上升 ，'D' 表示 下降 。

需要构造一个下标从 0 开始长度为 n + 1 的字符串，满足以下条件：

  • num 包含数字 '1' 到 '9' ，其中每个数字 至多 使用一次。
  • 如果 pattern[i] == 'I' ，那么 num[i] < num[i + 1] 。
  • 如果 pattern[i] == 'D' ，那么 num[i] > num[i + 1] 。

返回满足上述条件字典序 最小 的字符串 num。

  • 1≤pattern.length≤81 \le pattern.length \le 81≤pattern.length≤8

 

法一：全排列枚举

  1. 答案总个数不会超过 9!9!9!，循环里是常数级遍历，因此可以直接暴力
  2. 使用库函数 next_permutation

关于 next_permutation （在 algorithm 库里）：可以使用 o(n)o(n)o(n) 双指针实现，参见：31. 下一个排列 - 力扣（LeetCode）；与之相关的类似题有：556. 下一个更大元素 III - 力扣（LeetCode）

法二：贪心 o(n)o(n)o(n)

随便列举几个例子，会发现有趣的事：最优解会在下降的时候翻转
