654. 最大二叉树

给定一个不重复的整数数组 nums 。 最大二叉树 可以用下面的算法从 nums 递归地构建:

  1. 创建一个根节点，其值为 nums 中的最大值。

  2. 递归地在最大值 左边 的 子数组前缀上 构建左子树。

  3. 递归地在最大值 右边 的 子数组后缀上 构建右子树。

返回 nums 构建的 最大二叉树 。

  • 1≤nums.length≤10001 \le nums.length \le 10001≤nums.length≤1000

  • 0≤nums[i]≤10000 \le nums[i] \le 10000≤nums[i]≤1000

  • numsnumsnums 中的所有整数 互不相同

 

单调栈 o(n)o(n)o(n)

  1. 可以用 o(n2)o(n^2)o(n2) 暴力，也可以用RMQRMQRMQ o(nlogn)o(nlogn)o(nlogn)

  2. 实际上，本题的本质是笛卡尔树的构建，参考笛卡尔树 - OI Wiki (oi-wiki.org)

    笛卡尔树 = 堆 + 基于数组下标的二叉搜索树（中序遍历）

    对于一个节点，如果它大于左侧的所有点，则它作为新的树根，将左侧子树的根加到它的左边；否则找到第一个大于它的数xxx，将xxx对应子树的右子树作为它的左子树，它作为xxx的右子树

  3. 据此，只要使用栈来维护当前笛卡尔树的右链上的结点即可

2380. 二进制字符串重新安排顺序需要的时间

给你一个二进制字符串 s 。在一秒之中，所有 子字符串 "01" 同时 被替换成 "10" 。这个过程持续进行到没有 "01" 存在。

请你返回完成这个过程所需要的秒数。

  • 1≤s.length≤10001 \le s.length \le 10001≤s.length≤1000
  • s[i] 要么是 '0' ，要么是 '1' 。

 

一次遍历 o(n)o(n)o(n)

  1. 可以直接暴力，因为复杂度不会超过 o(n2)o(n^2)o(n2)

  2. 更优的做法是：

    把替换操作看成是1同时往左移动（如果左侧为 0 的话）。

    对于某一个1，如果左侧不为1，则移动时间等于左侧0的个数；如果左侧为1，则移动时间需要多加一秒，证明省略
