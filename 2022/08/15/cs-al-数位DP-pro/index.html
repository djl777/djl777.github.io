2376. 统计特殊整数

如果一个正整数每一个数位都是 互不相同 的，我们称它是 特殊整数 。

给你一个 正 整数 n ，请你返回区间 [1, n] 之间特殊整数的数目。

  • $1 \le n \le 2 \times 10^9$

 

记忆化搜索，数位DP模板

$f(i, mask, is_limit, is_num)$ 代表从左到右选到第 i 个数字时 ( i 从 0 开始)，前面数字已选状态为 mask 时的合法方案数。

  • mask：前面已择数字的状态

  • is_limit：当前是否受到了 n 的约束。若为真，则第 i 位填入的数字至多为 s[i]，否则可以是 9。如果在受到约束的情况下填了 s[i]，那么后续填入的数字仍会受到 n 的约束。

  • is_num：表示前面是否已经选择了数字，若为假，则当前位可以跳过（不填数字），或者要填入的数字至少为 1；若为真，则要填入的数字可以从 0 开始。

时间复杂度： 对于DP来说，等于状态个数×转移个数，$o(len(s)) = o(log(n))$, 再$\times 1024 \times 10$

思考：为什么其他语言并不需要记忆is_limit和is_num？

重复运算才需要记忆化，当is_limit为True时，并不会再重新访问这个转态，同理is_num，即在递归过程中只会用到一次
