门店商品调配 杭末专场

有数组 distributions，其中 distributions[i] = [from,to,num]，表示从 from 门店调配了 num 件商品给 to 门店。若要使得每一个门店最终借出和借入的商品数量相同，请问至少还需要进行多少次商品调配。

  • 1≤distributions.length≤81 \le distributions.length \le 81≤distributions.length≤8
  • 0≤fromi,toi<120 \le from_i, to_i < 120≤fromi​,toi​<12

题解

  1. 对于一个和为 0 长度为 n 的子集，它的最坏情况是移动 n - 1 次
  2. 如果在这个子集中找到两个和为 0 的子集，可以降低一次操作
  3. 枚举子集的技巧 for (int j = (i - 1) & i; j > 0; j = (j - 1) & i，如果想包含全集，那么从 j 开始，如果想包含空集，那么末尾特判（不能在循环里写，因为 -1 & i = i)
  4. 时间复杂度：元素个数为 i 的集合有 C(n,i) 个，其子集有 2i2^i2i 个，求和，使用二项式定理得 3n3^n3n
